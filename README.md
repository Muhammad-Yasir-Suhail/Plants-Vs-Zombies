# Plants vs Zombies (SFML - C++)

## 📌 Overview

This project is a **Plants vs Zombies-inspired game** developed using **C++** and **SFML**. It recreates the core mechanics of the original game, where players strategically place plants to defend against waves of zombies.

The project focuses on implementing game logic, rendering, event handling, and object-oriented design without using any game engine.

---

## 🚀 Features

* Real-time 2D gameplay using SFML
* Grid-based plant placement system
* Multiple plant types with unique behaviors
* Zombie spawning and movement system
* Collision detection (projectiles vs zombies)
* Health and damage mechanics
* Basic game loop (update, render, input handling)
* Win/Loss conditions

---

## ⚙️ Installation

1. Clone the repository:

```
git clone https://github.com/your-username/pvz-sfml.git
```

2. Navigate to the project folder:

```
cd pvz-sfml
```

3. Make sure **SFML is installed** and properly linked.

---

## ▶️ Run the Project

### Using IDE

* Open the project in your IDE
* Configure SFML include and library paths
* Build and run the main file 

---

### Using Command Line (g++)

```
g++ main.cpp -o game -lsfml-graphics -lsfml-window -lsfml-system
./game
```

---

## 🎮 Controls

* Mouse Click → Place plants 
* Keyboard (if used) → *1,2,3 etc. to select plants*
